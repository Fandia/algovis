# algovis
**algovis** is a program which can help you to visualize algorithms like a sorting or graphs traversing.

Currently implemented:
* Sorting:
  * Bubble sort
  * Quicksort
* Graphs:
  * Breadth-first search
  * Depth-first search

## Features
* Multilanguage
* Cross-platform(*PyQt*)

## Requirements
* Python 3
* PyQt 5
